## manual
1. 가상환경 생성 `$ python -m venv myvenv`
2. 가상환경 활성화
  Mac: `$ source myvenv/bin/activate`
  Windows: `$ . myvenv/Script/activate`
3. 필요한 패키지 설치 `$ pip install -r requirements.txt`
4. 모델 불러오기 `$ python manage.py migrate`
5.
